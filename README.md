### 🌌 What is Project Zenith?

Select **any location on Earth** and instantly visualize what's happening in the sky above — visible planets, satellites, the ISS, star charts, and celestial events — all in real time.

### 🚀 Features

| Feature | Tech |
|---------|------|
| 🗺️ **Leaflet Dark Map** | Location search with autocomplete |
| 🌍 **3D Interactive Globe** | Three.js with day/night terminator & atmosphere |
| 🛰️ **Real-Time Satellite Tracking** | satellite.js SGP4 propagation (10+ satellites) |
| ⭐ **D3 Star Chart** | Constellation lines & planet positions |
| 📡 **Satellite Pass Timeline** | D3-based next 6 hours visualization |
| 🌙 **Moon Phase & Events** | Live celestial event tracking |
